### what i do

I build things that **automate the tedious** and **help people do hard things**.

Currently building **[StreamLoop](https://streamloop.org)** — a coaching app for beginner streamers.
The core idea: becoming a public personality is a skill, not a talent. StreamLoop gives streamers a personalized loop of micro-tasks based on their specific weak spots — whether that's tech setup, content ideas, or just the fear of going live. Task → stream → report → analysis → better task. Repeat until you have an audience.
