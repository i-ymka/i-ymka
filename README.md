### what i build

I make things that **automate the tedious** and **help people do hard things**.

Currently building **[StreamLoop](https://streamloop.org)** — a coaching app for beginner streamers.
The idea: becoming a public personality is a skill, not a talent. StreamLoop gives streamers a personalized loop of micro-tasks based on their weak spots — whether it's tech setup, content ideas, or the fear of going live at all. Task → stream → report → analysis → better task. Repeat until you have an audience.
